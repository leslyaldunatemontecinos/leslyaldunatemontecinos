### 📈 Proyecto Destacado

#### 🎵 [Gestión de Canciones y Artistas](https://github.com/leslyaldunatemontecinos/Mostrar_Canciones_parte2)
Aplicación web robusta con arquitectura **MVC** completa utilizando **Spring Boot**, **Spring Data JPA** e **Hibernate**.
* **Arquitectura Completa:** Implementación de Modelos, Repositorios, Servicios y Controladores (`@Controller`, `@RestController`).
* **Operaciones CRUD:** Manejo de verbos HTTP (`GET`, `POST`, `PUT`, `DELETE`) para flujos completos de datos.
* **Persistencia y Modelado:** Diseño de Base de Datos relacional en MySQL conectando entidades mediante relaciones complejas **1:M (Uno a Muchos)**.
* **Validación y UX:** Renderizado dinámico de vistas con **JSP y JSTL**, control de excepciones y validación de formularios con `@Valid` y mensajes flash.
